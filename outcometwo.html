Trello

This website allows for web-based project management, it helps with the planning of projects on one platform.
It is easy to use as everything on there can be edited or drag and dropped like boards, cards and lists.
Trello is used for: Share files (incl. photos & videos) with your team members
Comment on a card to update workers
Keep track of to-do lists
Set coloured labels according to priority
Perform batch handling of cards in a list
Insert new cards; this is absolutely limitless

This is a tool that I really find amazing, it really helped me keep track of my work, helping keep to my deadlines.
In Unito Dos I am really happy with the tools I used however I believe I could have really used them for much more than what I actually did.

2.3. Strengths And Weaknesses

The overall project is a success as I have time to alter and optimise the website created to an adequate level. The satisfaction of getting an image into the correct location or getting the borders just the right size have made me feel engrossed with pleasure.

Strengths

The website shows different modes of which information can be displayed on a web page and how different types of data can also be displayed. Ultimately the website is functioning to the average standard. The moving background was a great achievement as I believe this is what really gave me an insight into how the code is written I had to do a lot of research to understand exactly how to generate the final product. My Unito Doser are mostly really happy with the end result which shows success. A well updated trello board which allows for good in time analysis of work done and to do.

Weaknesses

The background moves which can be seen as a distraction. The colour scheme is also a weakness as a few people of older age have admitted to difficulty when trying to read the text. There are very few pages and a lack of uniqueness to the webpages in comparison to others on the course. In addition I missed two days of this module during the start which has been a major drawback however the end result being a big rush to complete the project.
